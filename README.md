## 🚀 طريقة الاستخدام والبدء

للبدء في تصميم خريطتك الخاصة، اتبع الخطوات البسيطة التالية:

1. **التحميل:** قم بتحميل البرنامج من خلال الرابط المرفق بالأعلى لمشروع Arnis.
2. **التشغيل:** بعد فك الضغط والتثبيت، افتح الملف والمفروض أن تفتح معك خريطة تفاعلية.
3. **التطبيق:** من هناك، حدد منطقتك المفضلة، ثم **تابع الخطوات الموضحة بالفيديو** لإكمال عملية الاستخراج والتحويل بنجاح!
